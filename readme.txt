Taki zwyczajny saper
Mamy plansze z ukrytymi bombami.
Naszym zadaniem jest zaznaczyc ktore pola to bomby bez odkrywania ich
jesli odkryjemy pole z bomba gra sie konczy (sugestywny napis Game Over),
aby zagrac jeszcze raz trzeba ponownie uruchomic skrypt.
Kiedy wszystkie bomby zostana oznaczone pojawia sie napi You win!!!, 
aby zagrac jeszcze raz trzeba uruchomic jeszcze raz aplikacje.
Odkrywanie pol za pomoca lewego przycisku myszy, zaznaczanie bomb prawy przycisk.
